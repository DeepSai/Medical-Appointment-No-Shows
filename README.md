# Medical-Appointment-No-Shows
项目来源于kaggle，是我在进行**数据分析**初步学习之后的第一个练手项目。

在分析过程中发现，**病人能否按预约就诊**与**泰坦尼克号乘客能否生还**有相似之处，分析方法可以相互借鉴。
